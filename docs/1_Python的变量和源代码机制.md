## 内存地址
```python
var  = 1
id(var)
import sys
sys.getsizeof(var)
```
内存地址，和 js一样，值变量指向一块内存空间。

## 源码解析

- PyObject
- PyVarObject

对一个简单的值 `sys.getsizeof()` 会发现空间大很多。


PyObjectType 这个类型描述了很多元信息。内存地址、大约20多个内存指针等。所以占据了很大空间。
